4. OPERATOR PADA JAVASCRIPT : Aritmatika, Penugasan & Perbandingan

SJavaScript punya berbagai operator untuk mengolah data. Operator aritmatika digunakan untuk perhitungan seperti +, -, *, /, dan %. Operator penugasan dipakai untuk memberi atau memperbarui nilai variabel, contohnya =, +=, atau -=. Operator perbandingan digunakan untuk membandingkan dua nilai, seperti ==, ===, !=, <, >, <=, dan >=, hasilnya berupa true atau false.

5. OPERATOR PADA JAVASCRIPT : Logika, String, Typeof dan Kondisional

Operator logika dipakai untuk menghubungkan kondisi, seperti && (dan), || (atau), dan ! (tidak). Operator string digunakan untuk menggabungkan teks, biasanya dengan tanda +. Operator typeof berguna untuk mengecek tipe data dari sebuah nilai, misalnya number atau string. Sedangkan operator kondisional atau ternary (kondisi ? nilai1 : nilai2) dipakai untuk memilih hasil berdasarkan kondisi tertentu dalam satu baris kode.

6. TIPE DATA PADA JAVASCRIPT : STRING

String adalah tipe data untuk menyimpan teks di JavaScript. Teks ini bisa berupa huruf, angka, simbol, atau kombinasi semuanya, asalkan diapit oleh tanda kutip tunggal ('...'), kutip ganda ("..."), atau backtick (`...`). Dengan string, kamu bisa menampilkan pesan, menggabungkan teks, atau memanipulasi kata sesuai kebutuhan.
