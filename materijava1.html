1. Pengenalan Java

Java adalah bahasa pemrograman populer yang digunakan untuk pengembangan aplikasi desktop, mobile (Android), hingga aplikasi web. Java terkenal dengan prinsip "Write Once, Run Anywhere".

2. Variables in Java

Dalam Java, variabel berfungsi sebagai wadah untuk menyimpan data seperti angka, teks, atau karakter. Dengan variabel, kamu bisa mengolah dan mengatur data agar program berjalan sesuai kebutuhan.

3. Conditional Statements | If-else, Switch Break

Conditional statements digunakan untuk membuat keputusan dalam program. If-else dipakai ketika kamu ingin menjalankan kode tertentu hanya jika kondisi terpenuhi, sedangkan switch break membantu memilih salah satu dari banyak kemungkinan nilai dengan lebih rapi. Dengan conditional, program bisa berpikir dan menyesuaikan alurnya sesuai keadaan.
