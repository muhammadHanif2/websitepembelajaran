Belajar Konsep Dasar OOP (Object Oriented Programming) di Java

10. Arrays Introduction

OOP (Object Oriented Programming) adalah paradigma pemrograman yang berfokus pada objek. Objek memiliki data (atribut) dan perilaku (method).

11. 2D Arrays

2D array di Java adalah array yang punya baris dan kolom, mirip seperti tabel atau matriks. Struktur ini memudahkan kita menyimpan dan mengolah data yang berbentuk grid, misalnya papan permainan, data matematika, atau tabel angka.

12. Strings

String di Java adalah tipe data khusus yang digunakan untuk menyimpan dan mengolah teks. Dengan String, kamu bisa melakukan banyak hal seperti menggabungkan kata, menghitung panjang teks, atau mengambil karakter tertentu. String sangat penting karena hampir semua aplikasi butuh bekerja dengan teks.
