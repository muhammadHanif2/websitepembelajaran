6. FUNCTION

Function di PHP digunakan untuk mengelompokkan kode agar bisa dipakai ulang. Dengan function, kamu cukup menulis logika sekali lalu memanggilnya kapan saja dibutuhkan. Ini membuat program lebih rapi, terstruktur, dan mudah dikelola.

5. ARRAY

Array di PHP adalah variabel khusus yang bisa menyimpan banyak nilai sekaligus dalam satu nama. Dengan array, kamu bisa mengelola data lebih mudah, baik berupa daftar angka, teks, maupun pasangan kunci–nilai.

6. ASSOCIATIVE ARRAY

Associative array di PHP adalah array yang menggunakan nama kunci (key) sebagai pengganti indeks angka. Dengan cara ini, kamu bisa menyimpan dan mengakses data lebih jelas.
