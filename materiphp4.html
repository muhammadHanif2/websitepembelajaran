10. INSERT & DELETE

INSERT digunakan untuk menambahkan data baru ke dalam tabel di database, sedangkan DELETE dipakai untuk menghapus data yang sudah ada. Dengan bantuan PHP, perintah SQL ini bisa dijalankan langsung dari aplikasi web, sehingga pengguna bisa menambah atau menghapus data secara mudah.

11. SEARCHING

Searching digunakan untuk mencari data di dalam database berdasarkan kata kunci tertentu. Dengan PHP, kamu bisa membuat form pencarian lalu menggunakan perintah SQL agar hasil yang ditampilkan sesuai dengan input pengguna. Fitur ini sangat berguna untuk menampilkan data yang lebih spesifik dan cepat ditemukan.

12. UPDATE

UPDATE digunakan untuk mengubah data yang sudah ada di dalam tabel database. Dengan PHP, perintah SQL ini bisa dijalankan langsung dari aplikasi web, sehingga data lama dapat diperbarui sesuai kebutuhan tanpa harus menghapus atau menambahkan data baru.
