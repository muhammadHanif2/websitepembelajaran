7. TIPE DATA pada JAVASCRIPT : BOOLEAN

Boolean adalah tipe data yang hanya punya dua nilai: true atau false. Biasanya dipakai untuk kondisi logika, misalnya menentukan apakah sebuah pernyataan benar atau salah. Boolean sangat penting dalam percabangan (if-else) dan perulangan, karena menentukan alur jalannya program.

8. VARIABLE pada JAVASCRIPT

Variabel adalah wadah untuk menyimpan data di JavaScript. Dengan variabel, kita bisa menyimpan nilai seperti angka, teks, atau boolean, lalu memakainya lagi kapan saja. JavaScript punya tiga cara mendeklarasikan variabel, yaitu var, let, dan const. let dipakai untuk nilai yang bisa berubah, const untuk nilai tetap, sedangkan var adalah cara lama yang jarang dipakai di kode modern.

9. POPUP BOX pada JAVASCRIPT

JavaScript punya fitur popup box untuk berinteraksi dengan pengguna. Ada tiga jenis utama: alert() untuk menampilkan pesan sederhana, confirm() untuk meminta konfirmasi (OK atau Cancel), dan prompt() untuk meminta input dari pengguna. Popup ini biasanya dipakai untuk memberi informasi cepat atau menanyakan pilihan sebelum melanjutkan proses.
