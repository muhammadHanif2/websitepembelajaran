4. Output, Input, Variables, Operators & Conditionals

Di C++, kamu bisa menyimpan data menggunakan variabel, menampilkan data ke layar dengan output (cout), dan meminta data dari pengguna dengan input (cin). Dengan operator, kamu bisa melakukan perhitungan dan perbandingan, sedangkan conditional (seperti if dan else) membantu program mengambil keputusan berdasarkan kondisi tertentu. Dengan kombinasi ini, kamu bisa membuat program interaktif dan logis sesuai kebutuhan.

5. If Else & Switch Statement

FDi C++, If-Else digunakan untuk membuat keputusan berdasarkan kondisi: jika suatu kondisi benar, jalankan satu blok kode; jika salah, jalankan blok lain. Sedangkan Switch Statement digunakan untuk memilih di antara beberapa kemungkinan nilai dari sebuah variabel dengan lebih rapi dibanding banyak If-Else. Keduanya membantu program membuat keputusan yang berbeda sesuai data yang diberikan.

6. Loops in C++

Loop di C++ digunakan supaya program bisa berjalan berulang-ulang otomatis. For loop dipakai kalau kamu sudah tahu berapa kali pengulangannya. While loop jalan terus selama kondisinya benar, sedangkan Do-While loop dijalankan dulu minimal sekali sebelum kondisi dicek.
