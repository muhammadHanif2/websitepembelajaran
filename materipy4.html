10. Pengenalan String

String adalah tipe data untuk menyimpan teks, seperti kata, kalimat, atau karakter. Di Python, string ditulis di dalam tanda kutip tunggal '...' atau ganda "...". String bisa digabung dengan +, diulang dengan *, dan diakses per karakter menggunakan indeks. Dengan string, kamu bisa menyimpan nama, pesan, atau data teks lain dalam program dengan mudah.

11. Format String

Format string dipakai untuk menyisipkan nilai variabel ke dalam teks dengan rapi. Misalnya menggunakan f-string: f"Nama saya {nama}", atau metode .format(). Dengan format string, kita bisa membuat teks dinamis, menampilkan hasil perhitungan, atau menyusun pesan yang lebih mudah dibaca tanpa ribet menggabungkan banyak string dan variabel.

12. String width and Alignment

Di Python, kita bisa mengatur lebar tampilan string dan posisinya agar rapi, misalnya di tabel atau laporan. Gunakan metode seperti str.ljust(), str.rjust(), atau str.center() untuk menggeser teks ke kiri, kanan, atau tengah. Dengan pengaturan lebar dan alignment, tampilan output program jadi lebih rapi dan mudah dibaca.
