4. Menambahkan WordArt

Di Microsoft Word, kamu bisa menambahkan WordArt untuk membuat teks terlihat lebih kreatif dan menonjol. Fitur ini cocok digunakan pada judul, poster, atau dokumen yang memerlukan sentuhan desain yang menarik.

5. Menggabungkan File Microsoft Word Menjadi 1

Di Microsoft Word, kamu bisa menggabungkan beberapa file menjadi satu dokumen agar lebih rapi dan mudah dikelola. Fitur ini sangat berguna saat menggabungkan laporan, proposal, atau dokumen terpisah menjadi satu kesatuan.

6. Menambahkan Diagram

Di Microsoft Word, kamu bisa menambahkan diagram untuk visualisasi data.
