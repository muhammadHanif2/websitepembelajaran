10. CONTROL FLOW pada JAVASCRIPT

Control Flow adalah cara JavaScript mengatur jalannya kode, baris demi baris. Secara default, kode dijalankan dari atas ke bawah. Tapi dengan bantuan percabangan seperti if-else atau switch, serta perulangan seperti for, while, atau do-while, kita bisa mengatur alur program sesuai kondisi yang terjadi. Jadi, control flow ini ibarat peta jalan supaya kode berjalan sesuai logika yang kita mau.

11. PENGULANGAN : WHILE pada JAVASCRIPT

Perulangan while digunakan ketika kita ingin menjalankan kode selama kondisi tertentu masih benar (true). Selama syaratnya terpenuhi, blok kode di dalam while akan terus diulang. Kalau kondisi sudah salah (false), perulangan akan berhenti. while cocok dipakai kalau kita belum tahu pasti berapa kali perulangan harus dijalankan.

12. PENGULANGAN : FOR pada JAVASCRIPT

Perulangan for dipakai ketika kita sudah tahu berapa kali kode harus dijalankan. Struktur dasarnya terdiri dari tiga bagian: inisialisasi (nilai awal), kondisi (syarat berhenti), dan increment/decrement (perubahan nilai tiap loop). Dengan for, kita bisa mengulang sebuah perintah dengan rapi dan terkontrol, misalnya menampilkan angka 1 sampai 10.
