4. Casting Tipe Data

Casting adalah cara mengubah tipe data satu ke tipe data lain. Misalnya, angka bisa diubah jadi string supaya bisa digabung dengan teks, atau string "123" bisa diubah jadi angka untuk dihitung. Python punya fungsi bawaan seperti int(), float(), str(), dan bool() untuk melakukan casting. Dengan casting, kita bisa lebih fleksibel memanipulasi data sesuai kebutuhan program.

5. Operasi Aritmatika

Python bisa melakukan perhitungan matematika dasar seperti penjumlahan (+), pengurangan (-), perkalian (*), pembagian (/), sisa bagi (%), dan pangkat (**). Ada juga pembagian bulat (//) yang hasilnya dibulatkan ke bawah. Dengan operator aritmatika ini, kamu bisa menghitung angka, membuat kalkulator, atau memproses data numerik dalam program dengan mudah.

6. Operasi Komparasi

Operasi komparasi dipakai untuk membandingkan dua nilai dan menghasilkan True atau False. Contohnya == untuk sama dengan, != untuk tidak sama, > dan < untuk lebih besar atau lebih kecil, serta >= dan <= untuk lebih besar atau sama/kecil atau sama. Operator ini biasanya dipakai di kondisi if atau loop untuk mengambil keputusan dalam program.
