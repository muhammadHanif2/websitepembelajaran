1. PENGENALAN JAVASCRIPT

JavaScript adalah bahasa pemrograman yang berjalan di browser dan digunakan untuk membuat website interaktif. Hampir semua website modern menggunakan JavaScript.

2. NILAI DAN TIPE DATA PADA JAVASCRIPT

Di JavaScript, nilai bisa berupa angka, teks, boolean, objek, array, hingga fungsi. Semua nilai ini dikelompokkan dalam tipe data, seperti number, string, boolean, object, dan undefined. Dengan memahami tipe data, kamu bisa mengolah informasi dengan benar sesuai kebutuhan program.

3. TIPE DATA PADA JAVASCRIPT : ANGKA

Di JavaScript, tipe data Number dipakai untuk semua jenis angka, baik bilangan bulat maupun desimal. Angka bisa digunakan untuk perhitungan matematika seperti penjumlahan, pengurangan, perkalian, dan pembagian.
