1. PENGENALAN PYTHON

Python adalah bahasa pemrograman serbaguna yang mudah dipahami. Python digunakan dalam web development, data science, AI, hingga automasi.

2. VARIABEL PYTHON

Variabel di Python adalah wadah untuk menyimpan data. Kamu tidak perlu menentukan tipe data saat membuat variabel, cukup langsung isi nilainya. Misalnya, kalau kamu tulis nama = "Hanif", otomatis Python tahu itu string. Kalau umur = 20, Python paham itu angka. Variabel bisa menyimpan berbagai tipe data seperti angka, teks, boolean, hingga list. Intinya, variabel memudahkan kita memberi nama pada data biar bisa dipanggil lagi tanpa harus menulis ulang nilainya.

3. Tipe Data

PTipe data di Python adalah kategori untuk jenis data yang bisa disimpan dalam variabel. Contohnya ada angka (int, float), teks (str), benar/salah (bool), serta kumpulan data seperti list, tuple, dan dictionary. Dengan tipe data, Python tahu cara memproses setiap nilai. Misalnya, angka bisa dihitung, string bisa digabung, dan boolean bisa digunakan di kondisi if.
