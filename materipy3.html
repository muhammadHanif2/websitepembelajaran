7. Operasi Logika atau Boolean

Operator logika dipakai untuk menggabungkan atau membalik kondisi yang bernilai True atau False. Contohnya and (keduanya harus True), or (salah satu True), dan not (membalik nilai True jadi False atau sebaliknya). Operator ini sering digunakan di if atau loop untuk membuat keputusan yang lebih kompleks dalam program.

8. Operator Bitwise

Operator bitwise bekerja langsung pada representasi biner angka. Contohnya & (AND), | (OR), ^ (XOR), ~ (NOT), serta << dan >> untuk geser bit kiri atau kanan. Operator ini berguna untuk manipulasi data tingkat rendah, optimasi, atau pemrograman yang butuh operasi biner seperti kriptografi dan kontrol hardware.

9. Operator Assignment

Operator assignment dipakai untuk memberi nilai ke variabel, biasanya menggunakan =. Selain itu, ada juga kombinasi dengan operasi lain seperti +=, -=, *=, /=, dan %= supaya lebih ringkas. Misalnya x += 5 sama dengan x = x + 5. Dengan operator ini, kita bisa memperbarui nilai variabel dengan cepat tanpa menulis ulang nama variabelnya.
